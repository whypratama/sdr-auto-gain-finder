# SDR Auto Gain Finder

Automatic optimal gain finder for RTL-SDR + Bias Tee + LNA setups used in SatNOGS and amateur satellite ground stations.

## Features

- Auto gain scanning
- Bias Tee compatible
- Overload detection
- Noise floor analysis
- SatNOGS friendly

## Installation

```bash
sudo apt install rtl-sdr python3-pip python3-venv
```

## Run

```bash
python3 auto_gain.py
```
